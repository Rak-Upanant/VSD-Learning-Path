🎯 บทเรียนที่ 12: คำอธิบายการควบคุมแบบ Vector (Field Oriented Control)

การควบคุมแบบ Vector หรือที่รู้จักกันในชื่อ Field Oriented Control (FOC) แสดงถึงความก้าวหน้าที่สำคัญเหนือกว่าการควบคุมแบบ Scalar (V/Hz) ในเทคโนโลยี VSD ในขณะที่ Scalar Control ให้การควบคุมความเร็วในระดับพื้นฐาน แต่ Vector Control ให้ประสิทธิภาพที่เหนือกว่า โดยเฉพาะอย่างยิ่งในด้านการควบคุมแรงบิด, การตอบสนองแบบไดนามิก และประสิทธิภาพ

หลักการของการควบคุมแบบ Vector

เป้าหมายของการควบคุมแบบ Vector คือการควบคุมมอเตอร์ AC ให้เหมือนกับว่ามันเป็นมอเตอร์ DC ในมอเตอร์ DC แรงบิดและฟลักซ์แม่เหล็กจะถูกควบคุมอย่างอิสระโดยกระแสอาร์เมเจอร์และกระแสสนามแม่เหล็กตามลำดับ Vector Control บรรลุการแยกส่วนที่คล้ายกันในมอเตอร์ AC โดยการควบคุมกระแสสเตเตอร์ในกรอบอ้างอิงที่หมุนไปพร้อมกับฟลักซ์ของโรเตอร์ สิ่งนี้ทำให้สามารถควบคุมแรงบิดและฟลักซ์ได้อย่างอิสระ ส่งผลให้ประสิทธิภาพไดนามิกและประสิทธิภาพการทำงานดีขึ้น

การแปลงพิกัด (Coordinate Transformations)

หัวใจของการควบคุมแบบ Vector อยู่ที่การแปลงพิกัด การแปลงเหล่านี้จะแปลงกระแสสเตเตอร์สามเฟส (Ia, Ib, Ic) ให้อยู่ในกรอบอ้างอิงที่หมุนได้สองมิติ (d-q frame) โดยแกน d จะอยู่ในแนวเดียวกับฟลักซ์ของโรเตอร์ และแกน q จะตั้งฉากกับแกน d การแปลงนี้ทำให้ VSD สามารถควบคุมส่วนประกอบของกระแสที่สร้างฟลักซ์ (Id) และกระแสที่สร้างแรงบิด (Iq) ได้อย่างอิสระ

การควบคุมกระแส (Current Control)

เมื่อกระแสสเตเตอร์ถูกแปลงเป็นกรอบอ้างอิง d-q แล้ว VSD จะสามารถควบคุม Id และ Iq ได้อย่างอิสระ โดยทั่วไปจะใช้ตัวควบคุมแบบ Proportional-Integral (PI) เพื่อควบคุม Id และ Iq ให้ได้ตามค่าอ้างอิง

  • การควบคุม Id: ตัวควบคุม Id จะควบคุมฟลักซ์ของโรเตอร์ ค่าอ้างอิง Id_ref โดยทั่วไปจะถูกตั้งค่าให้รักษาฟลักซ์คงที่เพื่อประสิทธิภาพสูงสุด
  • การควบคุม Iq: ตัวควบคุม Iq จะควบคุมแรงบิดของมอเตอร์ ค่าอ้างอิง Iq_ref จะเป็นสัดส่วนโดยตรงกับแรงบิดที่ต้องการ

ตัวอย่างการใช้งาน

แขนหุ่นยนต์ (Robotics Arm): แขนหุ่นยนต์ต้องการการควบคุมมอเตอร์ข้อต่ออย่างแม่นยำ Vector Control ใช้เพื่อควบคุมแรงบิดของมอเตอร์ ทำให้หุ่นยนต์สามารถเคลื่อนที่ตามเส้นทางที่แม่นยำและรักษตำแหน่งได้แม้ภายใต้โหลดที่เปลี่ยนแปลงไป

รถยนต์ไฟฟ้า (EV): รถยนต์ไฟฟ้าใช้ Vector Control เพื่อควบคุมมอเตอร์ลากจูง ให้การเร่งที่ราบรื่น, การใช้พลังงานอย่างมีประสิทธิภาพ และการเบรกแบบปั่นไฟกลับ (Regenerative Braking)

เครื่องจักรการผลิตความแม่นยำสูง: เครื่องจักรเช่น เครื่องกัด CNC และเครื่องกลึงต้องการการควบคุมมอเตอร์แกนหมุน (Spindle) ที่แม่นยำอย่างยิ่ง Vector Control ช่วยรักษความเร็วและแรงบิดของแกนหมุนให้คงที่แม้ภายใต้ภาระการตัดที่แตกต่างกัน

สถานการณ์สมมติ: ระบบลิฟต์

ลองนึกภาพลิฟต์ในอาคารสูง ลิฟต์ต้องให้การเดินทางที่ราบรื่นและสะดวกสบายสำหรับผู้โดยสารโดยไม่คำนึงถึงน้ำหนักบรรทุก

VSD ที่ใช้ Vector Control จะควบคุมแรงบิดและความเร็วของมอเตอร์ได้อย่างแม่นยำ โดยปรับค่า Iq_ref เพื่อชดเชยน้ำหนักบรรทุกที่เปลี่ยนแปลงไปในขณะที่ผู้โดยสารเข้าและออกจากลิฟต์ ทำให้ลิฟต์สามารถเร่งและลดความเร็วได้อย่างนุ่มนวล

🏭 การประยุกต์ใช้จริงที่โรงงาน AA

โรงงาน AA มีปัญหากับระบบสายพานลำเลียงที่ควบคุมด้วย V/Hz Drive แบบเดิม ซึ่งมีการไหลของวัสดุที่ไม่สม่ำเสมอและสายพานหยุดทำงานเป็นครั้งคราวเมื่อมีโหลดหนัก

  • ปัญหา: V/Hz Drive ไม่สามารถควบคุมแรงบิดได้ดีพอ
  • วิธีแก้ปัญหา: เปลี่ยนไปใช้ VSD ที่มีการควบคุมแบบ Vector ซึ่งสามารถควบคุมแรงบิดได้อย่างแม่นยำ ทำให้การไหลของวัสดุสม่ำเสมอและป้องกันการหยุดทำงาน
  • ประโยชน์: การไหลของวัสดุดีขึ้น, ลดเวลาหยุดทำงาน, เพิ่มประสิทธิภาพการผลิต และยังประหยัดพลังงานได้ดีขึ้นด้วย

📝 แบบฝึกหัด

  1. การคำนวณการแปลงพิกัด: กำหนดให้กระแสสเตเตอร์สามเฟสคือ Ia = 10A, Ib = -5A, Ic = -5A และมุมฟลักซ์ของโรเตอร์ θ = 30 องศา จงคำนวณหากระแส d-q (Id และ Iq)
  2. การวิเคราะห์การประมาณค่าฟลักซ์: เปรียบเทียบข้อดีและข้อเสียของวิธีการวัดฟลักซ์โดยตรงและการประมาณค่าฟลักซ์โดยอ้อม
  3. การปรับจูนตัวควบคุมกระแส: อธิบายว่าค่าเกนของ PI controller (Kp และ Ki) ส่งผลต่อประสิทธิภาพของตัวควบคุม Id และ Iq อย่างไร

🏁 สรุปและขั้นตอนต่อไป

ในบทเรียนนี้ เราได้สำรวจหลักการของการควบคุมแบบ Vector ซึ่งเป็นวิธีการควบคุมมอเตอร์ AC ที่ซับซ้อนและให้ประสิทธิภาพที่เหนือกว่า Scalar Control เราได้หารือเกี่ยวกับการแปลงพิกัด, เทคนิคการประมาณค่าฟลักซ์ และกลยุทธ์การควบคุมกระแส

ในบทเรียนถัดไป เราจะเจาะลึกถึง Direct Torque Control (DTC) ซึ่งเป็นวิธีการควบคุมขั้นสูงอีกวิธีหนึ่งที่ให้แนวทางที่แตกต่างในการควบคุมมอเตอร์สมรรถนะสูง การทำความเข้าใจทั้ง Vector Control และ DTC จะทำให้คุณมีความเข้าใจที่ครอบคลุมเกี่ยวกับเทคนิคการควบคุมขั้นสูงที่ใช้ใน VSD สมัยใหม่
